# rag-llm-poc

## POC Title: DocuTalk Assist - Conversational Document Query with Speech Interaction
## Requirements:
* Chat with documents (using audio)
1. User should be able upload documents (can be pdf or text files) and ask queries using documents as information source.
2. User will speak to provide its queries and AI would generate response in audio as well i.e Speech-to-Text & Text-to-Speech.
3. UI should allow user to upload documents and start or see conversation in text.
4. Use any LLM api  - like gemini or Open AI to handle conversation.
